# powerco-customer-churn-analysis
Developed an end-to-end customer churn prediction solution for PowerCo using Python, EDA, feature engineering, and Random Forest classification. Analyzed customer price sensitivity, identified key churn drivers, evaluated model performance, and provided data-driven business recommendations to improve customer retention.
# PowerCo Customer Churn Prediction

## Project Overview

This project was completed as part of the BCG Data Science Virtual Experience Program.

The objective was to investigate customer churn at PowerCo and determine whether customer price sensitivity is a major driver of customer attrition.

The project follows a complete data science workflow:

- Exploratory Data Analysis (EDA)
- Feature Engineering
- Churn Prediction using Random Forest
- Business Recommendations

---

## Business Problem

PowerCo observed customer churn and suspected that customers were leaving due to increased sensitivity to energy prices.

The goal was to:

- Identify churn drivers
- Build a predictive model
- Generate actionable business recommendations

---

## Methodology

### 1. Exploratory Data Analysis

- Data type inspection
- Missing value analysis
- Distribution analysis
- Correlation analysis
- Customer churn exploration

### 2. Feature Engineering

Created new features including:

- Price sensitivity
- Average yearly price changes
- Average six-month price changes
- Customer tenure
- Consumption ratios
- Profitability ratios

### 3. Machine Learning

Model Used:

- Random Forest Classifier

Evaluation Metrics:

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

---

## Key Findings

- Price-related variables contribute significantly to churn prediction.
- Customer consumption and tenure also influence churn behavior.
- The model can identify customers at high risk of leaving.
- Targeted retention campaigns can reduce customer attrition.

---

## Business Recommendations

1. Implement churn prediction in customer retention workflows.
2. Focus retention efforts on high-risk customers.
3. Offer targeted incentives before contract renewal periods.
4. Continuously retrain the model as customer behavior evolves.
